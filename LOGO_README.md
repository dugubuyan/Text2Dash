# Text2Dash Logo 使用说明

## Logo 文件位置

### 前端应用
- **完整 Logo**: `frontend/public/logo.svg` (128x128)
- **Favicon**: `frontend/public/favicon.svg` (32x32)

### 官网
- **完整 Logo**: `website/logo.svg` (128x128)
- **Favicon**: `website/favicon.svg` (32x32)

## Logo 设计说明

### 设计理念
- **蓝色渐变背景**: 代表科技感和专业性（#1890ff → #096dd9）
- **动态柱状图**: 象征数据分析和可视化能力，带有动画效果
- **AI 闪光元素**: 体现智能AI驱动的特性

### 特点
- 现代简洁的设计风格
- 带有动画效果，更具活力
- SVG 格式，支持任意缩放
- 适配深色和浅色背景

## 使用场景

1. **网站 Favicon**: 浏览器标签页图标
2. **应用 Logo**: 页面头部、登录页面
3. **营销材料**: 宣传册、PPT、海报
4. **社交媒体**: 头像、封面图

## 技术规格

- 格式: SVG (矢量图)
- 完整版尺寸: 128x128px
- Favicon 尺寸: 32x32px
- 圆角: 完整版 60px，Favicon 6px
- 主色调: #1890ff (Ant Design 蓝)
